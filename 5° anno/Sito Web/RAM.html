LA RAM

La memoria RAM (Random Access Memory), chiamata anche memoria principale, viene utilizzata per caricare i programmi che devono andare in esecuzione.

Alloggia negli slot della scheda madre. Le RAM che vengono usate attualmente sono le SDRAM (Synchronous Dynamic RAM).

Le RAM possono essere di diversi tipi: DDR, DDR2, DDR3 e DDR4. Le RAM DDR (Double Data Rate) hanno un doppio canale, quindi un doppio trasferimento dati. Più alto è il numero dopo "DDR" più alto sarà la velocità di trasferimento dei dati.
